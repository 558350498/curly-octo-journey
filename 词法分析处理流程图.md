# 词法分析处理流程图

下面给出与 `实验1-参考代码.txt` 对应的词法分析流程（Mermaid）。

```mermaid
flowchart TD
    A[开始] --> B[打开 example.txt]
    B -->|失败| C[输出 error 并结束]
    B -->|成功| D[读取1个字符到 cbuffer]

    D --> E{cbuffer == EOF?}
    E -->|是| Z[输出 over 并结束]
    E -->|否| F{isalpha(cbuffer)?}

    F -->|是| G[alphaprocess\n读取字母数字串]
    G --> H{是否关键字?}
    H -->|是| I[输出 关键字 (1,idx)]
    H -->|否| J[加入/查询标识符表\n输出 标识符 (2,idx)]
    I --> D
    J --> D

    F -->|否| K{isdigit(cbuffer)?}
    K -->|是| L[digitprocess\n读取无符号整数串]
    L --> M[加入/查询常数表\n输出 常数 (3,idx)]
    M --> D

    K -->|否| N[otherprocess]
    N --> O{算术运算符 +-*/ ?}
    O -->|是| P[输出 运算符 (4,idx)]
    P --> D

    O -->|否| Q{关系运算符起始符号?}
    Q -->|是| R[超前读取1字符\n优先匹配 <= >= !=]
    R --> S[输出 运算符 (4,idx)]
    S --> D

    Q -->|否| T{是否 ':=' ?}
    T -->|是| U[输出 运算符 (4,10)]
    U --> D
    T -->|否| V{是否界符 , ; { } ( ) ?}
    V -->|是| W[输出 界符 (5,idx)]
    W --> D
    V -->|否| X[若非空白/换行\n输出 error]
    X --> D
```
